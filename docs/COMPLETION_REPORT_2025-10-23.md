## 🔧 사용된 기술

### CSS clamp() 함수
```scss
padding: clamp(2rem, 4vh, 3.5rem);
/* 최소 32px, 화면 높이의 4%, 최대 56px */
```

### max-height 제한
```scss
.hero-section-small {
  min-height: clamp(350px, 40vh, 500px);
  max-height: 60vh; /* 과도한 확장 방지 */
}
```

### @media (min-height) 쿼리
```scss
@media (min-height: 1080px) {
  .hero-section-small {
    max-height: 550px !important;
  }
}
```
